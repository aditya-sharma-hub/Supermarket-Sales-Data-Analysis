# 🛒 Supermart Grocery Sales – Retail Analysis

This project explores **retail grocery sales data** to generate actionable business insights.  
Using **Python, Pandas, Matplotlib, Seaborn, and scikit-learn**, the project covers **EDA, visualization, and predictive modeling** to better understand sales drivers and forecast performance.  

The dataset represents **customer orders from a fictional grocery delivery platform in Tamil Nadu, India.**

---

## 📊 Project Overview

- **Domain:** Retail / Sales Analytics  
- **Tools Used:** Python, Pandas, NumPy, Matplotlib, Seaborn, scikit-learn, SQL, Excel  
- **Project Type:** Data Analytics + Machine Learning  
- **Dataset Source:** [Supermart Grocery Sales Dataset](https://drive.google.com/file/d/1Vx-Ibn11HKofkJasjMZFyigemSu7TOeB/view?usp=sharing)  
- **Difficulty Level:** Intermediate  

---

## 🧩 Problem Statement

Retail companies face challenges in identifying profitable product categories, understanding customer buying patterns, and forecasting sales.  
This project aims to:  
- Explore **sales and profit trends** across categories, sub-categories, and cities.  
- Understand the **impact of discounts** on profitability.  
- Identify **top-performing regions and cities**.  
- Build a **machine learning model** to predict sales.  

---

## 🧠 Key Objectives

✔️ Perform **data cleaning and preprocessing** (missing values, duplicates, data types).  
✔️ Analyze **sales by category, sub-category, and region**.  
✔️ Track **monthly and yearly sales trends**.  
✔️ Visualize **discount vs profit relationships**.  
✔️ Build and evaluate a **Linear Regression model** for sales prediction.  
✔️ Provide **business insights** for decision-making.  

---

## 🗃️ Dataset Features

| Column            | Description |
|-------------------|-------------|
| `Order ID`        | Unique identifier for each order |
| `Customer Name`   | Name of the customer |
| `Category`        | Broad product category (e.g., Beverages, Food Grains) |
| `Sub Category`    | More specific category |
| `City`            | Customer city |
| `Order Date`      | Date of order placement |
| `Region`          | Region within Tamil Nadu |
| `Sales`           | Sales revenue |
| `Discount`        | Discount applied |
| `Profit`          | Profit earned |
| `State`           | State of customer |

---

## 📈 Key Insights

- 🥚 **Egg, Meat & Fish** contributed ~15% of total sales, making it the top category.  
- 📆 **2017 & 2018** accounted for more than 50% of sales, showing strong yearly growth.  
- 🌍 **Top Cities** contributed disproportionately to total revenue.  
- 🎯 **Discounts increased sales but reduced profitability**—requiring balanced pricing strategy.  
- 🤖 Linear Regression achieved **R² ≈ 0.82**, showing good predictive performance for sales forecasting.
